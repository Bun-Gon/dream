Functional Notation

If A, B are sets, let f: A → B.
That is, f is a function from A to B.
Let f : x ↦ y
That is, y is the image of x under f.
Then y is denoted by f(x).
In other words, the image of x under f is denoted by f(x).
It is read: "f of x".
For example, Let J be the set of all integers and let f : J → J be the function such that f : x ↦ x2 + x + 1.
In our functional notation, we write
f(x) = x2 + x + 1.
which is read : " f of x is x2 + x + 1".
It means "the image of x under f is x2 + x + 1" or the value of f(x) is x2 + x + 1".
The expression f(x) = x2 + x + 1 is called the formula of the function f.
The formula for the function is the convenient way to dewscribe the rule which tells us what the image of each element of the domain is.

Solution: f(x) = x2 + x + 3

f(-3) = (-3)2 + (-3) + 3
Note: We find that f maps both 2 and -3 to 9. Therefore it is not a one-to-one function.

Equality of function

Two functions f and g are equal (and we write f = g) if and only if
(1) f and g have the same domain,
(2) f and g have the same codomain, and
(3) f(x) = g(x) for each element x of the domain.
Thus, f : A → B and g : A → B are regarded as the same function if and only if f(x) = g(x) for each x in A.

Example 5.
Let A = {1} and B = {1, 2}.
Let f : A → B be defined by f(x) = x2 and
Let g : A → B be defined by g(x) = 2x - 1. Then
f(1) = 12 = 1
g(1) = 2(1)-1 = 1
∴ f(1) = g(1), and so f(x) = g(x) for every x in A. ∴ f = g

Example 6.
Let C = {1, 2} and D = {1, 3, 4}.
Let f : C → D be defined by f(x) = x2 and
Let g : C → D be defined by g(x) = 2x - 1. Then
f(1) = 12 = 1, g(1) = 2(1)-1 = 1.
f(2) = 22 = 4, g(2) = 2(2)-1 = 3.
f(2) ≠ g(2).
Therefore we have f ≠ g.
