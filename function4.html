Some ideas on Functions

In this section, we will study more about functions. First we will review some ideas on functions:
Let A and B be sets. A function f from A to B is a relation in which each element of A is related to exactly one element of B. We write f : A → B.
If an element x ∈ A is related to the element y ∈ B, we say that "y is the image of x under f" and we write
f(x) = y
we also write f : x ↦ y.
If f is a function from A to B then the set A is called the domain of the function and the set of all the images of element of A is called the range of the function.
We will write Rf (A) to denote the range of f.
Thus Rf (A) = {f(x) | x ∈ A}.

Example 1.
Let A = {1, 2, 3, 4} and B = {2, 4, 6, 8, 10}. Let the function f : A → B be defined by f(x) = 2x. Find the range of f.
Solution
We can write down the image of each of the element of A as follows:
f(x) = 2(x)
f(1) = 2(1) = 2
f(2) = 2(2) = 4
f(3) = 2(3) = 6
f(4) = 2(4) = 8
The range of the function f is
Rf (A) = {2, 4, 6, 8}
In this example, the range is not exactly B itself. Rf (A) is proper subset of B.

Example 2.
Let A = {-1, 0, 1} and B = {0, 1}
Let the function g : A → B be defined by g(x) = x2. Find the range of g.
Solution
The images of elements of A are
g(x) = (x)2
g(-1) = (-1)2 = 1
g(0) = (0)2 = 0
g(1) = (1)2 = 1
The range of the function g is
Rg (A) = {0, 1} and Rg (A) = B.
In this case, the range of g is B itself.
We notice that more than one element of A correspond to 1 ∈ B.

One - to - One correspondence

Let f : A → B be function. If each element of B is related to exactly one element of A, then f is called a one-to-one correspondence between A and B. The sets A and B are said to be in one-to-one correspondence.

Example 3
Let A = {1, 2, 3} and B = {3, 6, 9}.
Let the function f : A → B be defined by f(x) = 3x.
f(1) = 3(1) = 3
f(2) = 3(2) = 6
f(3) = 3(3) = 9
We have
f : 1 ↦ 3
f : 2 ↦ 6
f : 3 ↦ 9
We find that
3 ∈ B is related to 1 ∈ A only
6 ∈ B is related to 2 ∈ A only
9 ∈ B is related to 3 ∈ A only
This function is a one-to-one correspondence between A and B.
